# Chat App

A simple WhatsApp-style chat application built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Chat sidebar
- Message bubbles
- Send messages
- Enter-to-send
- Automatic replies
- Mobile-friendly layout

## Files

- `index.html` - Main page
- `style.css` - Styling
- `script.js` - Chat functionality

## How to Run

1. Download the project files.
2. Open `index.html` in a web browser.

## Technologies Used

- HTML5
- CSS3
- JavaScript
